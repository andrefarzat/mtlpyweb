Event Calendar

Nous avons maintenant un calendrier Google pour les événements MP! Vous pouvez faire une recherche pour le calendrier sur Google Calendar, mais il est aussi incorporé ici: http://montrealpython.org/event-calendar/ Où vous pourrez voir un bouton pour s'y abonner facilement.

Si vous utilisez un autre logiciel, vous pouvez vous abonner au calendrier MP avec les liens suivants:

* xml * ical * ou simplement voir le calendrier sur une page complète

Nous nous efforcerons de garder ce calendrier à jour, incluant les liens vers les annonces des événements, leur date et heure, et leur endroit. Ce calendrier sera la source d'information à jour pour savoir ce qu'il se passe avec Montréal-Python!
