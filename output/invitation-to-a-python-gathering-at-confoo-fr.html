Invitation to a Python gathering at Confoo

Bonjour Pythoneux

Ce vendredi, 12 mars 2010 à 9:00, dans le cadre de Confoo cette semaine, un rassemblement non-officiel montreal-python est organisé. Le tout a lieu dans au stand Savoir-Faire Linux, dans la salle St-Pierre, en face de la salle de diner/lunch. (Hôtel Hilton Bonaventure)

Les sujets traités: - Intro à Python pour les nouveaux - Pylons, Django, TG2 - Packaging, distutils2, .deb, .rpm, .egg - Documentation avec Python, Sphinx et autres

Vous pouvez vous joindre à nous même si vous n'êtes pas inscrit à à la conférence Confoo à partir de 13h30 ! Wow !
